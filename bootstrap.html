What is BOOTSTRAP?

Bootstrap is a free and open-source tool collection for creating responsive websites and web applications. It is most popular HTML, CSS, JavaScript framework for developing responsive, mobile-first web sites. It solves many problems which we had once, one of which is the cross-browser comptability issue. Nowdays,the websites are perfect for all the browsers(IF, Firefox and Chrome and for all sizes of screens (Desktop, Tablets, Phablets and Phones. All thanks to Bootstrap deveopers- Mark Otto and Jacob Thornton of Twitter, though it was later declared to be an open-source project.
