What is HTML?

The Hyper Text Markup Language or HTML is the standard markup language for documents designed in a web browser. It can be assisted by the technologies such as cascading style sheets (CSS) and scripting languages as JavaScript.HTML is used to create electronic documents (called pages) that are displayed on the World Wide Web. Each page contains a series of connections to other pages called hyperlinks. Every web page you see on the Internet is written using one version of HTML code or another.
